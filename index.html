Rutas

Frontend

Frontend Developer es la persona responsable de lo que se puede ver de un sitio web, menús, imágenes, botones, etc. Así como consultar APIs.

  • Terminal, VSC y Git/Github
  • *UI/UX
  • HTML
  • CSS
  • Sass
  • Bootstrap o Tailwind
  • JavaScript y/o TypeScript
  • React o Angularo Vue
  • Consultar APIs

Backend

Backend Developer es la persona responsable de escribir el código que se ejecuta del lado del servidor, maneja base de datos. Así como crear APIs.

  • VSC, Git/Github y Servidores
  • Base de Datos
  • *Algoritmos y Estructuras de Datos
  • JavaScript y/o TypeScript
  • Node o Django o Laravel o Spring Boot
  • Construir APIs
  • Patrones de Arquitectura

Fullstack

Full Stack Developer es la unión de Frontend y Backend Developer, es la persona responsable de todo el desarrollo de un sitio web.

  • VSC, Git/Github y Servidores
  • *UI/UX
  • HTML
  • CSS
  • Sass
  • Bootstrap o Tailwind
  • Base de Datos
  • *Algoritmos y Estructuras de Datos
  • JavaScript y/o TypeScript
  • React o Angularo Vue
  • Node o Django o Laravel o Spring Boot
  • Consultar y Construir APIs
  • Patrones de Arquitectura
